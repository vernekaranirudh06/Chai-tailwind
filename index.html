Why ChaiCSS

Everything you need,
nothing you don't.

⚡
Zero Dependencies
Pure vanilla JS. No build step, no npm, no config. Just drop chai.js in your HTML and go.
🧠
Smart Parser
Intelligent class name parsing handles spacing, colors, typography, layout, borders and more from a single utility map.
🎨
Utility-first
Inspired by Tailwind, built from scratch. Write chai-p-16 and get padding: 16px — instantly.
📐
Layout Support
Flex, grid, block, inline, hidden — full display control with directional flex utilities like chai-flex-col-reverse.
🔤
Typography
Control font size, weight, line height, letter spacing, text alignment and color with intuitive class names.
📦
Lightweight
The entire engine is under 100 lines of clean, readable JavaScript. Easy to understand, extend and contribute to.
