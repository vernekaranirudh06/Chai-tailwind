Getting Started

Up and running
in 60 seconds

No npm. No build tools. No config. Just one script tag and you're styling.

1
Download or copy chai.js
Grab the ChaiCSS engine file and place it in your project folder.
# Your project structure my-project/ ├── index.html └── chai.js ← place it here
↓ Download chai.js
2
Include the script in your HTML
Add the script tag at the bottom of your <body>. This ensures the DOM is ready before ChaiCSS scans it.
<!DOCTYPE html> <html> <head> <title>My App</title> </head> <body> <!-- your content here --> <script src="chai.js"></script> <!-- ← at the bottom --> </body> </html>
3
Start using chai-* classes
Add chai-* class names to any HTML element. ChaiCSS will scan, parse and apply styles automatically.
<div class="chai-p-24 chai-bg-#111 chai-rounded-12"> <h1 class="chai-text-32 chai-fontWeight-700 chai-text-white"> Hello, ChaiCSS! ☕ </h1> <p class="chai-mt-12 chai-text-16 chai-text-gray"> Zero config. Just classes. </p> </div>
💡 Tip: ChaiCSS reads classes at page load. The script tag must be at the bottom of body so all elements exist in the DOM when it runs.
