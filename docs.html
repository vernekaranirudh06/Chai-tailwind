ChaiCSS Reference

Complete reference for all supported utility classes and how the engine works.

Introduction
ChaiCSS is a utility-first CSS engine built in vanilla JavaScript. Instead of writing traditional CSS, you write class names like chai-p-16 or chai-bg-red and ChaiCSS applies the corresponding inline styles to your elements.
<!-- Without ChaiCSS --> <div style="padding:16px; background:red; border-radius:8px"> ... </div> <!-- With ChaiCSS --> <div class="chai-p-16 chai-bg-red chai-rounded-8"> ... </div>
How It Works
When the script loads, ChaiCSS performs these steps:
// 1. Scans ALL DOM elements const elements = document.querySelectorAll("*") // 2. Filters elements that have a class attribute const filtered = Array.from(elements).filter(el => el.hasAttribute('class')) // 3. For each element, finds classes starting with "chai-" // 4. Parses the class: splits by "-" to extract type + value // 5. Looks up type in classUtility map → gets CSS property // 6. Applies inline style to the element
Class Syntax
All ChaiCSS classes follow this pattern:
chai-{type}-{value} Examples: chai-p-16 → padding: 16px chai-bg-red → background-color: red chai-text-center → text-align: center chai-flex → display: flex (no value needed)
Numeric values — automatically get px appended.
String values — passed as-is (e.g. color names, CSS keywords).
No value — for layout classes like chai-flex, chai-hidden.

ClassCSS PropertyExample

Typography
Control all text-related styles.
ClassCSS PropertyExample
chai-text-{n}font-sizechai-text-24 → font-size: 24px
chai-text-{align}text-alignchai-text-center, chai-text-left
chai-fontWeight-{n}font-weightchai-fontWeight-700 → font-weight: 700
chai-fontSize-{n}font-sizechai-fontSize-18 → font-size: 18px
chai-leading-{n}line-heightchai-leading-24 → line-height: 24px
chai-tracking-{n}letter-spacingchai-tracking-2 → letter-spacing: 2px
Note: chai-text-* is smart — it detects alignment keywords (left, right, center, justify, start, end), numeric values (font-size), and everything else is treated as a color.
